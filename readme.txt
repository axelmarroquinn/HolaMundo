*** INSTALACIÓN ***
1. Copiar o mover la carpeta INVENTARIO_DISEÑOS_SA al servidor local o remoto.
2. Crear una Base de Datos en MySQL con el nombre de preferencia, importando
   la base de datos ubicada en la carpeta DB del proyecto.
3. Abrir PHP > main.php, y configurar los datos del servidor en la función 'conexión'.
4. Correr el programa.
